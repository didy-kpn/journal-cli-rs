## journal-cli-rs

日記作成および管理のためのRust製のCLIツールです。

## インストール

```
$ cargo install --git https://github.com/didy-kpn/journal-cli-rs.git
```

## 使い方

### オプションの説明

オプションについては以下のコマンドを実行してください。

```
$ journal-cli --help
```
